Frequently Asked Questions

What's the difference between Stack 2.9 and other AI assistants?

Stack 2.9 learns from every interaction. While other assistants reset between sessions, Stack 2.9 remembers, evolves, and gets better over time.

How does self-evolution work?

Stack 2.9 observes its problem-solving process, extracts patterns from successful solutions, stores learned knowledge in persistent memory, and applies that knowledge to future tasks.

Can I self-host Stack 2.9?

Absolutely! Stack 2.9 is designed for self-hosting. Deploy with Docker, run locally, or integrate into your own infrastructure. Your data stays yours.

What tools are available?

37 built-in tools including file operations, code execution, search, API calls, git operations, terminal commands, and more. All accessible via natural language.

Is it free to use?

Yes! Stack 2.9 is open-source under Apache 2.0 license. You can use, modify, and distribute it freely.

What hardware do I need?

Stack 2.9 runs on consumer hardware. A machine with 16GB+ RAM can handle the 32B model. GPU acceleration recommended for best performance.
